# 🥗 EatWise

AI-powered weekly meal planner for families, generated by Claude.

## Features

- **AI-generated weekly plans** – Claude creates 3 meals per day (Mon–Sun) tailored to your family
- **Fresh cooking** – Ingredient amounts shown per adult, with optional child variation
- **Shopping multiplier** – Each meal shows recipe view (1 adult) and scaled shopping quantities (× number of people)
- **Week selection** – Generate a plan for the current or upcoming calendar week
- **Archive** – All past plans accessible via the sidebar
- **Admin UI** – Configure family size, calorie targets, dietary preferences, and dislikes
- **Docker-ready** – Plans and context are persisted via a volume mount

## Quickstart

```bash
cp .env.example .env
# Fill in your values (see below)
npm install
npm run dev
```

App runs at [http://localhost:3000](http://localhost:3000)

## Environment Variables

```env
ANTHROPIC_API_KEY=sk-ant-...   # Anthropic API key (console.anthropic.com)
ADMIN_PASSWORD=yourpassword    # Password for /admin (any username works)
```

## Docker

```bash
docker compose up -d
```

Data is persisted via the volume mount `./data:/app/data` — `context.json` and all generated plans survive container restarts.

## Admin UI

Accessible at `/admin` — log in with any username and the `ADMIN_PASSWORD` from your `.env`.

Configurable fields:
- Number of adults and children
- Calorie targets per group
- Dietary requirements (e.g. "one adult vegetarian")
- Food preferences and dislikes

## Plan Format

Each plan is stored as `data/plans/KWxx_yyyy.json`. Meal ingredients are always specified **per 1 adult**. The child variant (`child`) is only included if it differs from the adult portion.

```jsonc
{
  "week": "KW10_2026",
  "generatedAt": "2026-03-07T22:30:00Z",
  "days": [
    {
      "day": "Montag",
      "meals": [
        {
          "type": "Frühstück",
          "name": "Bircher Müsli",
          "description": "Overnight oats with apple and berries",
          "adult": { "kcal": 380, "ingredients": ["80g rolled oats", "200ml oat milk"] },
          "child": { "kcal": 280, "ingredients": ["60g rolled oats", "150ml oat milk"] }
        }
      ]
    }
  ]
}
```

## Deployment

GitHub Actions builds a Docker image on every push to `main`, pushes it to `ghcr.io`, and deploys via SSH.

Required repository secrets:

| Secret | Description |
|--------|-------------|
| `ANTHROPIC_API_KEY` | Anthropic API key |
| `ADMIN_PASSWORD` | Admin UI password |
| `SSH_HOST` | Server hostname or IP |
| `SSH_USER` | SSH username |
| `SSH_KEY` | SSH private key |

Server setup:

```bash
mkdir ~/eatwise && cd ~/eatwise
# Place docker-compose.yml and .env here
```
